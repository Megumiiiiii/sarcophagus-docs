# **導入**

サルコファガス・デッドマンズ・スイッチ（DMS）は、ユーザーが無力化され、死亡したり、データにアクセスできなくなった場合のデータの回復や受け渡しの問題に対する分散型の解決策を提供します。その目的は、データを暗号化されたコンテナ（サルコファガス）に格納し、作成者が何らかの方法で対話できなくなった場合にのみ、意図された受信者によってアクセスできるようにすることです。一度DMSがトリガーされると、データは受信者の秘密鍵を使用して復号化できます。統合により、このプロセスをバックエンドツールとして使用できるようになり、さまざまなdAppsやアプリケーションに対応します。これにより、パスワード、個人文書、財務記録、秘密鍵、さらには一部の旧式のアプリケーションやサービスを含む、機密データとデジタルアセットの安全な回復と移転が不変に確保されます。

# 一般的なユースケースの概要

### デジタル・リビング・ウィル

サルコファガスは、多くのユースケースに対応できる汎用ツールとして設計されましたが、最も一般的に言及されるユースケースは、デジタルワールドでの相続/データの受け渡しです。この目的のため、サルコファガスは、暗号化されたデジタル・リビング・ウィルを作成し、チェーン上に保存することを可能にします。

デジタル・リビング・ウィルは、彼らの意志を伝える手段がなくなった場合の医療治療の選択肢を概説する文書であることがあります。この場合、指定された受信者は文書にアクセスし、患者の意向が尊重されていることを確認できます。また、パスワード、メールアカウント、秘密鍵、その他の機密データと共に、所有者がそれらを管理できなくなった場合の指示を含めるためにも使用できます。これにより、受信者または相続人が会計を実行したり、メールとソーシャルアカウントを管理したりするために必要な情報にアクセスできるようになります。重要なデータでは不変の保証が重要であり、サルコファガスはこれらのデジタル遺産を将来の世代のために保護するための安全で信頼性のあるDMSソリューションを提供します。

### 緊急時のコミュニケーション

野外でのハイカーが外部との通信手段を持たない状況を想像してみてください。彼らは緊急連絡先情報を保管し、近い家族などの指定された連絡先がそれにアクセスし、その情報を公園警備員や捜索救助チームに中継できるサルコファガスDMSを作成できます。ハイカーが無力化されたり、通信できなくなった場合、この受信者は情報にアクセスし、彼らの愛する人々にすぐに連絡を取ることができ、すべての関係者に安心感を提供します。

このテクノロジーは、通信線がダウンしたり信頼性が低い自然災害のような緊急事態でも使用できます。ユーザーはサルコファガスに緊急連絡先情報を保存することで、自分自身が通信できない場合でも愛する人々に緊急事態が発生したことが通知されることを確信できます。

このテクノロジーにより、ユーザーは予期せぬ状況に備え、愛する人々が危機の際にケアを受けることを安心して知ることができます。

- [ハリケーン・カトリーナ（2005年）](https://en.wikipedia.org/wiki/Hurricane_Katrina)：アメリカの湾岸地域を襲った後、多くの地域が停電や通信インフラのない状態になりました。緊急対応者は努力を協力して調整し、多くの人々が助けを呼び出したり状況について情報を得たりすることができませんでした。
- 海上の緊急事態の場合、船舶は通常、[Search and Rescue Transponders（SART）](https://en.wikipedia.org/wiki/Search_and_rescue_transponder)を使用して、近くの船舶や捜索救助チームが受信できる救難信号を送信します。しかし、SARTが機能しない場合や信号を効果的に送信できない場合があります。サルコファガスDMSは、位置データを定期的にブロックチェーンに送信することにより、単一障害点によって妨害されないバックアップ通信システムとして機能できる可能性があります。

### 政治活動とジャーナリズム

政治活動家とジャーナリストはしばしば危険な環境にさらされています。彼らの命だけでなく、彼らの情報源の命も危険にさらされています。彼らの安全性を向上させるためには、ソーシャルメディアの情報源、連絡先、研究などの機密情報を保管する必要があります。ここでサルコファガスが役立ちます。指定された受信者がアクセスできるようになる重要なデータを安全に保管する環境を提供します。ジャーナリストまたは活動家に何かが起こった場合、その作業は中断せずに続行できます。これにより、真実が明らかにされ、正義が維持されます。さらに、データのバックアップを定期的に作成するためにも使用でき、活動家とジャーナリストはデータの信頼性のあるバックアップを後で回復できることを安心して知ることができます。これは国境越えの場合にも便利であり、データがこれらのシナリオで没収されないようにするのに役立ちます。

- [プレスエンブレム](https://www.pressemblem.ch/casualties.shtml)によると、2013年以来、1130人以上のジャーナリストとメディア関係者が死亡しています。
- [ジャーナリスト保護委員会](https://cpj.org/reports/2020/12/record-number-journalists-jailed-imprisoned/#:~:text=In%20its%20annual%20global%20survey,high%20of%20272%20in%202016.)によれば、2020年12月1日にジャーナリズム活動に関連して少なくとも274人のジャーナリストが投獄され、2016年の最高記録を超えました。
- [Top10VPN](https://www.top10vpn.com/research/cost-of-internet-shutdowns/)は、インターネットシャットダウンと政府によるインターネットアクセスの拒否を社会的なコントロール手段として追跡しています。現在、政府の検閲名目で420億ドル以上の経済的コストが発生しています。

### パスワードと資格情報の回復

サルコファガスDMSの別の潜在的なユースケースは、忘れられたパスワードや資格情報の回復です。ユーザーは、サルコファガスにアクセスできるのはユーザー自身だけである復旧情報（電子メールのパスワードやセキュリティの質問など）を保存できます。これにより、ユーザーはハッキングやソーシャルエンジニアリング攻撃に対して脆弱性のある従来の回復方法に依存せずにアカウントを安全に回復できるようになります。

**復旧統合のシナリオ例:**

[Gnosis Safe](https://safe.global/): セーフの署名者は、1人の署名者が署名の能力を失った場合にアクセスを保護するためにDMSを使用できます。これは、別のアドレスに署名権を渡す統合型のバックアップであるか、現在のアドレスのPKを受信者にファイルで送信するだけのものです。

[Twitter](https://twitter.com/)または他のソーシャルメディアのユーザーは、アカウントのアクセス回復オプション機能としてDMSを使用できます。これにより、中央集権型サーバーではなく、信頼性のある実行構造が提供されます。

**統計データ:**

- [Chainalysisの調査](https://blog.chainalysis.com/reports/bitcoin-market-data-exchanges-trading/)によれば、2017年と2020年の調査によれば、現在のビットコインの約20%が紛失しており、その多くは紛失した秘密鍵に帰因しています。
- パスワードのリセット/回復の電子メールは、ユーザーの代替電子メールアドレスに送信されることがよくあります。特定の場合、サーバーがダウンしている場合、回復用の電子メールが使用されていない場合、または回復用の電子メールも侵害されている場合、これは信頼性が低い場合があります。

### タイムカプセル

タイムカプセルは、思い出や工芸品を将来の世代のために保存する方法です。Arweaveブロックチェーンの持続性のおかげで、Sarcophagus DMSは重要な文書、写真、ビデオを将来の世代が「開封」できるように保存するためのタイムカプセルとして使用できます。これにより、将来の世代はこの情報にアクセスし、自分たちの家族の歴史について学んだり、DAOタウンシップがデジタルの歴史的なコレクションを残すことができるかもしれません。また、その暗号化機能のおかげで、内容は時間が来るまで秘密のままです。長期保存の可能性は無限です！

**統計:**

- 最も有名なタイムカプセルのいくつかには、1940年に封印され、8113年まで封印されることを意図した[Crypt of Civilization](https://en.wikipedia.org/wiki/Crypt_of_Civilization)と、1939年および1964年のワールドフェアで埋められ、それぞれ6939年と8113年に開封予定のWestinghouse Time Capsulesが含まれています。
- タイムカプセルは一般的に将来の特定の日に開封することを意図していますが、失われたり忘れられたりする事例もありました。一部の場合、建設作業や発掘作業中に偶然発見されたタイムカプセルもあります。
- 物理的なタイムカプセルが特定の期間無傷で保持されることを保証する確実な方法はなく、侵食、湿度、気温の変化などの要因がすべて内容物の保存に影響を与える可能性があります。

### ノード/オペレーターの評判

Sarcophagusネットワークでは、ノードの評判は、ユーザーが自分のサルコファガスを監視するためにどのノードを選ぶかを選択するのに役立つ重要な要素です。ノードの評判システムは、成功したアンラップと失敗したアンラップ、および悪意のある振る舞いの告発に基づいてノードのパフォーマンスを表示するように設計されています。これらの指標は、ユーザーがネットワーク内で最も信頼性のあるノードを特定するのに役立ちます。

サルコファガスを効果的に保護するためのトラックレコードを披露することにより、ノードオペレーターは、Sarcophagusネットワークまたはオペレーターのサービスを技術的な別の方法で使用することに興味を持つ可能性のある潜在的なユーザーやパートナーにその信頼性と能力を示すことができます。

ノードオペレーターがこの情報をソーシャルメディアチャンネルで共有したり、独自のマーケティング資料に含めたりすることで、この情報を活用する別の方法も考えられます。彼らは潜在的なユーザーやパートナーに対する主要なセールスポイントとしてその信頼性を強調することができます。また、この情報を既存のネットワーク接続との信頼構築に活用することもでき、これによりリピートビジネスと自然な紹介が生まれる可能性があります。

ノードの評判統計は、ユーザーとノードオペレーターの両方にとって、Sarcophagusネットワークのユースケースと価値提案の重要な要素です。
