# **介绍**

石棺死人开关（DMS）为用户在用户变得无能、死亡或无法访问其数据的情况下提供了一种分散的数据恢复或传递问题的解决方案。其目的是将数据存储在一个加密容器（石棺）中，只有在创作者以某种方式无法互动时才能被预期的接收者访问。一旦DMS被触发，数据可以使用接收者的私钥进行解密。集成将使这个过程可以作为各种dApps和应用程序的后端工具来使用。这将不可变地确保敏感数据和数字资产的安全恢复和转移，包括密码、个人文件、财务记录、私钥，甚至一些传统的应用程序和服务。

# 一般用例概述

### 数字生前遗嘱

石棺被设计为一个通用工具，可以服务于许多用例，但最常引用的用例是数字世界中的继承/数据传递。为此，石棺使用户能够创建一个加密的数字生前遗嘱（直到发布时保持秘密），并存储在链上。

数字生前遗嘱可以是一份文件，概述了一个人在他们变得无能或无法表达他们的意愿的情况下对医疗治疗的偏好。在这种情况下，指定的接收者可以访问文件并确保患者的愿望得到尊重。它还可以用于存储重要信息，如密码、电子邮件帐户、私钥和其他敏感数据，以及在所有者无法再管理它们的情况下传递的指令。这确保了接收者或继承人能够访问所有必要的信息来进行会计或管理电子邮件和社交帐户。对于重要数据，不可变的保证至关重要，石棺提供了一个安全和无信任的DMS解决方案，以保存这些数字遗产供后代使用。

### 紧急通信

想象一种情景，一个徒步旅行者迷失在荒野中，无法与外界通信。他们可以创建一个石棺DMS来存储紧急联系信息，可以被指定的联系人访问，如亲近的家人，然后被转达给公园管理员或搜救队。如果该徒步旅行者变得无能或无法交流，接收者可以访问信息并迅速联系他们的亲人，为所有相关方提供安心。

这项技术还可以在其他紧急情况下使用，例如飓风或地震等自然灾害，通信线路可能中断或不可靠。通过将紧急联系信息存储在石棺中，用户可以放心，即使他们无法自行通信，也会通知他们的亲人在紧急情况下，他们的亲人将会被通知。

借助这项技术，用户可以为意外情况做好准备，并放心知道他们的亲人在危机时将得到照顾。

- [卡特里娜飓风（2005年）](https://en.wikipedia.org/wiki/Hurricane_Katrina)：它袭击了美国墨西哥湾沿岸后，许多地区都没有电力或通信基础设施。紧急救援人员努力协调他们的工作，许多人无法寻求帮助或获取有关情况的信息。
- 在海上发生紧急情况时，船只通常使用[搜索与救援测波器（SARTs）](https://en.wikipedia.org/wiki/Search_and_rescue_transponder)来传输紧急信号，这个信号可以被附近的船只或搜救队接收。然而，有些情况下SART可能无法正常运行或无法有效地传输信号。通过定期将位置数据传输到区块链上，石棺DMS可以充当备份通信系统，不能被单点故障干扰。

### 政治活动和新闻工作

政治活动家和记者经常置身于危险的环境中。他们的生命以及他们的消息来源的生命都处于危险之中。为了提高安全性，他们需要存储敏感信息，如消息来源、联系人和研究。这就是石棺派上用场的地方。它提供了一个安全的环境来存储关键数据，可以被指定的接收者访问。如果记者或活动家发生了什么事情，他们的工作可以继续进行而没有任何中断。这确保了真相会被揭示，正义会得以维护。此外，它可以定期创建数据备份，使活动家和记者可以安心知道他们有可靠的数据备份以供以后恢复使用。在跨越边界的情况下，这也很方便，以确保数据不会在这些情况下被没收。

- 据[新闻徽章](https://www.pressemblem.ch/casualties.shtml)称，自2013年以来已经发生了1130多起记者和媒体工作者伤亡事件。
- [保护记者委员会](https://cpj.org/reports/2020/12/record-number-journalists-jailed-imprisoned/#:~:text=In%20its%20annual%20global%20survey,high%20of%20272%20in%202016.)发现，截至2020年12月1日，在与他们的工作有关的情况下，至少有274名记者被监禁，超过了2016年272名的最高记录。
- [Top10VPN](https://www.top10vpn.com/research/cost-of-internet-shutdowns/)拥有一个追踪器，用于追踪政府以社会控制的方式封锁互联网和限制互联网访问的情况。目前，他们已经追踪到超过420亿美元的经济损失，以政府审查的名义。

### 密码和凭证恢复

Sarcophagus DMS的另一个潜在用例是恢复丢失或遗忘的凭证。用户可以将他们的恢复信息，如电子邮件密码或安全问题，存储在只能由用户访问的石棺中。这将确保用户有一种安全的方式来恢复他们的帐户，而不依赖于可能容易受到黑客或社会工程攻击的传统恢复方法。

**示例恢复集成方案：**

[Gnosis Safe](https://safe.global/)：Safe上的签署者可以使用DMS来保护访问权限，以防其中一名签署者失去签署能力。这可以是一个集成备份，将签署权传递给另一个地址，或者只是将当前地址的PK以文件形式发送给接收者。

[Twitter](https://twitter.com/)或其他社交媒体用户可以将DMS用作丢失帐户访问恢复的恢复选项功能。这将提供一个无需信任的执行结构，而不是使用中央服务器进行恢复选项。

**统计数据：**

- [Chainalysis的研究](https://blog.chainalysis.com/reports/bitcoin-market-data-exchanges-trading/)在2017年和2020年发现，大约20%的现有比特币流通量已经丢失，其中相当大一部分归因于丢失的私钥。
- 密码重置/恢复电子邮件通常发送到用户的备用电子邮件地址，在某些情况下，如果服务器宕机、恢复电子邮件未使用或恢复电子邮件也被入侵，这可能不可靠。

### 时间胶囊

时间胶囊是保留记忆和文物供后代使用的一种方式。由于Arweave区块链的永久性，Sarcophagus DMS可以用作时间胶囊，用于存储重要文件、照片和视频，以供后代“打开”。这确保了后代可以访问这些信息，并了解他们的家庭历史，或者DAO城镇可以留下数字历史收藏。此外，由于它的加密功能，内容将保密直到合适的时间。长期存储的可能性是无限的！

**统计数据：**

- 一些最著名的时间胶囊包括[文明的隐匿之地](https://en.wikipedia.org/wiki/Crypt_of_Civilization)，它在1940年封存，打算保持封存直到8113年，以及西属时光囊，它们埋在1939年和1964年的世界博览会上，分别计划在6939年和8113年打开。
- 尽管时间胶囊通常打算在未来的特定日期打开，但也有一些遗失或被遗忘的情况。在某些情况下，时间胶囊是在施工或挖掘工作期间意外发现的。
- 没有一种确保物理时间胶囊在特定时间段内保持完好无损的方法，因为诸如侵蚀、湿度和温度变化等因素都会影响内容的保存。

### 节点/操作员声誉

在Sarcophagus网络中，节点声誉是一个重要的方面，它帮助用户选择要监视其石棺的节点。节点声誉系统旨在显示节点的性能，根据成功和失败的解封以及对恶意行为的指控来评估。这些指标帮助用户确定网络中最可靠的节点。

通过展示他们有效保护石棺的记录，节点操作员可以向有兴趣使用Sarcophagus网络或操作员的服务的潜在用户或合作伙伴展示他们的可靠性和能力。

节点操作员可以潜在地利用这些信息的另一种方式是在他们的社交媒体渠道上分享它，或者将其包含在他们自己的营销材料中。他们可以将可靠性作为潜在用户或合作伙伴的关键卖点。他们还可以利用这些信息建立与现有网络连接的信任，这可能会导致再次合作和有机推荐。

节点声誉统计数据对于Sarcophagus网络的用例和价值主张对用户和节点操作员都是一个重要方面。
